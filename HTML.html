Definition of html language

Text encoding languages:

Markup language , or, as defined in the text markup language , is a set of hints, commands, and tags about a specific text; it defines how it is organized, formatted, and described, and has recently been used in character-line systems over the Internet.

One of the most famous examples of coding languages is the HTML language, which is among the important Internet protocol, and this language uses a number of famous methods in publishing to form a link between the author and editor.

HTML language:

HyperText Markup Language , referred to as HTML , is a special coding language used for the purpose of designing and creating Web pages, and we can say that it is the main structure of web pages and infrastructure. It provides a detailed description of how You will have the mechanism of displaying the contents of the website and be divided into a title and paragraphs, all this depending on what is known as the tag.

Tags:

It is important to note that tags are the main part of building a web page. By using the page, the page takes special commands and executes them to be used in the formation of the web page. Usually, all tags are enclosed between <>, and break into two main types:

Start tag: This requires the designer to place each mark or start and end command, to be executed correctly, and the starting tag is usually set as follows:/.

.End tag: The end tag is usually accompanied by the /, sign for closing the given command, and is as follows:/.

HTML work:

This is a key condition for starting a Web page. All tags must be enclosed between these two tags, as are required for the Ms-front page or notepad . The main purpose of starting and ending the work with the above two terms is to give the web browser an order to translate the commands into content for the end user of the webpage and its visitors.

All tags that contain commands and formats to be applied to the contents of the page are placed in text, tables, hyperlinks, drawings and formats within this field before reaching the closing of this tag.

It should be noted that it requires the web page designer to use this language to save his work in html format so that he can apply all the contents of the file based on this language. We can also control the size, color, font type and direction using special tags.
