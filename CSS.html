Definition of CSS language

What is CSS?

CSS is the acronym for Cascading Style Sheet. It uses CSS styles in the design of web pages to control the shape of the entire site in terms of type, color, or size of the font without having to type or repeat these codes.

Each page within the site, and thus can be through one file only make any adjustments to all files, saving a lot of effort and time. The function of this technique is to control how the web page is displayed without interfering with the content, which makes it easier to manage the site, modifying one CSS file will affect the design of all the pages of the site. CSS files simplify site design and management and solve many of the problems a developer may encounter.

For example, if you have designed your site using HTML, assuming that this site consists of five pages, You need to re-design the five pages again, this is very easy and you have to start now. Assuming that your site contains more pages and has 50 or 100 pages, you will definitely If you have designed your site from the very beginning using CSS, all you have to do is click on the link below Edit only one file.

CSS fonts eliminate the need to use lots of HTML commands, such as the FONT command that specifies the type, size, or color of a font, as well as the TABLE command used to create tables.

In other words, anyone can use CSS to design their entire site without using tables as is followed In traditional ways of designing pages, this does not mean that tables will not be used with CSS, tables have another function is to view and format data that need to be in a table. W3C has created this technology and made it a standard for web design, and many sites have begun to move from the traditional style of design using tables to the new style using CSSfor web design, and many sites have begun to move from the traditional style of design using tables to the new style using CSS.

Noticeable :

W3C is the abbreviation of the World Wide Web consortium and is the organization responsible for issuing standard versions of HTML.

Why use CSS style sheets?

First, it is currently an agreed standard, standards make things easier and CSS technology will be supported in the near future by all IT companies, making them widely used and therefore Web page designers and developers learn from now to be ready for the future.

Second, because the CSS stylesheets will be compatible with future hardware and software, they will be used in many areas that may appear in the future and can not be predicted now.

Third: the separation of content from the design, and this leads to simplify the process of managing the site, because the design can be controlled from a single file, can add and modify some elements in the design of the site without the need to modify all the files of the site.

Fourthly, making your site files smaller, reducing operating costs, and helping your site visitor, regardless of the speed of his or her Internet connection, call website pages through his browser of any kind faster than before.
